Rust 之旅

今天开始 Rust 的旅程。^_^

安装 Rust

我的系统是 deepin20，直接根据官网安装教程走就行了，使用 Rustup 安装

curl –proto ‘=https’ –tlsv1.2 -sSf https://sh.rustup.rs | sh

查看 Rust 版本

rustc -V

rustc 1.45.2 (d3fb005a3 2020-07-31)

更新 Rust

rustup update
