位运算

  2021-04-18   计算机   位运算 0  评论 字数统计: 1318字 阅读时长: 3分

什么是二进制

二进制是计算机内部运算中采用的进制，在这样的进制系统下，只有0，1两个数字，计算机内部的所有运算（包括位运算）都是在二进制的基础上进行的。

位运算分类

基本的位运算分为6种，分别为按位与、按位或、按位异或、按位取反，左移和右移。

运算 运算符 解释
按位与 & 两个对应位都为1时，结果才为1，否则为0
按位或 ` `
按位异或 ^ 两个对应位相同为0，不同为1
按位取反 ~ 0变1，1变0
左移 << 各二进位全部左移若干位，高位丢弃，低位补0
右移 >> 各二进位全部右移若干位，对无符号数，高位补0，有符号数，使用最高位的数补齐

什么是原码、反码和补码及其计算方式

原码、反码和补码都是计算机中有符号数的表示方法，都由符号位和数值位组成。符号位用0表示正数，1表示负数。

原码

最高位表示符号位，其它位存放该数的二进制的绝对值。

0表示正数，1表示负数。用来区分正负数。

将结果转为十进制，明显正负数相加结果是不正确的。这是由于符号位导致的。

为了解决一个数的正负相加不等于零这个问题，引入了 反码。

反码

正数的反码等于原码，负数的反码是它的原码除符号位外，其余位按位取反。

-10 1000 1010 1111 0101

虽然解决了一个数的正负相加不等于零这个问题，但是引入了新的问题，两个负数相加的结果不正确

补码

计算方式：正数和0的补码等于原码，负数的补码是其反码加一。

原码 补码

-10 1000 1010 1111 0110
-5 1000 0101 1111 1011

---------

---------

# 模：表示n位的计算机计量范围是`0-2^n-1`，模为`2^n`
# 这里用的是8位，所以模为 2^8=256
# 10-5=10+256-5=261 ==> 1 0000 0101 ==> 0000 0101 ==> 5

应用场景

1、判断奇偶

    println!("{}", 11 & 1 == 1); // true
    println!("{}", 18 & 1 == 0); // true

2、交换两个数

    let mut a = 10;
    let mut b = 5;
    a = a ^ b;
    b = a ^ b; // b = (a ^ b) ^ b = a ^ (b ^ b) = a ^ 0 = a
    a = a ^ b; // a = (a ^ b) ^ a = (a ^ a) ^ b = 0 ^ b = b 
    println!("a = {}, b = {}", a, b); // a = 5, b = 10
    // 两个相同的数异或的结果为0，一个数和0的异或结果是这个数本身。

3、找出没有重复的数

给出一个整型数组，[0,5,8,9,5,8,0]，其中有一个数只出现了一次，其他的数都出现了两次，找出出现一次的数。

    let arr = [0, 5, 8, 9, 5, 8, 0];
    let mut temp = arr[0];
    let mut i = 1;
    while i < arr.len() {
        temp ^= arr[i];
        i += 1;
    }
    println!("{}", temp); // 9

    // 0 ^ 5 ^ 8 ^ 9 ^ 5 ^ 8 ^ 0 = (0 ^0) ^ (5 ^ 5) ^ (8 ^ 8) ^ 9 = 0 ^ 0 ^ 0 ^ 9 = 9
