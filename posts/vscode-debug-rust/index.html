如何在 VSCode 中调试 Rust

如何在 VSCode 中调试 Rust

来源：如何在 VSCode 中调试 Rust 代码

  • 安装 CodeLLDB
  • 创建 launch.json 文件，添加配置如下
{
  "version": "0.2.0",
  "configurations": [
    {
      "name": "(Linux) Launch",
      "type": "lldb",
      "request": "launch",
      "program": "${workspaceRoot}/target/debug/文件名称",
      "args": [],
      "cwd": "${workspaceRoot}"
    }
  ]
}
