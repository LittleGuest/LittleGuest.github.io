如何使用Docker搭建MySQL服务

如何使用Docker搭建MySQL服务

下载 MySQL 镜像

官方镜像的 MySQL 版本是8.x

$ docker pull mysql

启动容器

  • -d 在后台运行
  • –name mysql 指定容器的名字为 mysql
  • -p 映射容器ip:3306到主机ip:13306
  • -e MYSQL_ROOT_PASSWORD=root 初始化root用户密码为root
$ docker run -d --name mysql -p 13306:3306 -e MYSQL_ROOT_PASSWORD=root mysql

如果要将数据保存在主机上，这需要加上下面的参数

  • -v 将容器中的文件挂载到主机
$ docker run -d --name mysql -p 13306:3306 -e MYSQL_ROOT_PASSWORD=root -v /home/mysql/cnf.d:/etc/my.cnf.d -v /home/mysql/data:/var/lib/mysql/data mysql

进入容器

$ docker exec -it mysql bash

进入 MySQL

$ mysql -u root -p

输入密码，能正常进入，就说明已经搭建好了。

连接 MySQL

这里注意使用 Navicat15 连接时，主机地址为 localhost 时，会出现如下错误：

2002 - Can't connect to local MySQL server through socket '/var/lib/mysql/mysql.sock' (2 "没有那个文件或目录")

解决方案之一 将 localhost 换成 127.0.0.1

而我使用 DBeaver 连接时，则没有出现这种情况。
