FastJson 使用 toJSONString 时字段为空问题

首先来看看代码

Map<String, Object> map = new HashMap<>(2);
map.put("reportNo", null);
map.put("reportDate", new Date());
System.out.println(JSON.toJSONString(map));

输出结果为

{"reportDate":1611756215215}

为什么会出现这种情况呢？

通过分析源码得知

if (value == null) {
    if (!SerializerFeature.isEnabled(out.features, features, SerializerFeature.WriteMapNullValue)) {
        continue;
    }
}

当值为 null 时，默认是不会被序列化的，但是我们有时候是需要的，那么如何解决呢？

解决方法

通过刚才的源码得知，可以指定 SerializerFeature.WriteMapNullValue 来序列化为 null 的值。

Map<String, Object> map = new HashMap<>(2);
map.put("reportNo", null);
map.put("reportDate", new Date());
System.out.println(JSON.toJSONString(map, SerializerFeature.WriteMapNullValue));

输出结果为

{"reportDate":1611757294826,"reportNo":null}
